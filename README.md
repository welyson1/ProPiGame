# Perguntas ProPiGame

Uma breve descrição sobre o que esse projeto faz e para quem ele é


## Passos para contribuir
Edite o arquivo quizQuestionsPI_perguntas_final.json

```JSON
...
    "Q_ID0": "Quais destas é uma indicação geográfica?",
    "A0_ID0": "Queijo Canastra",
    "A1_ID0": "Playstation",
    "A2_ID0": "Rio Amazonas",
    "A3_ID0": "Harry Potter",
    "S_ID0": 0,
    "txtS_ID0": "O Queijo Canastra é um tipo de queijo brasileiro, de origem e produção de Minas Gerais, na região da Serra da Canastra. É uma indicação geográfica",
...
```
Descrição dos campos

__Q_ID0:__ Pergunta

__A0_ID0:__ Questão 1

__A1_ID0:__ Questão 2

__A2_ID0:__ Questão 3

__A3_ID0:__ Questão 4

__S_ID0:__ Indice da questão correta

__txtS_ID0:__ Explicação

Basta substituir pelas suas questões e interar o index de questões _ID(sequencia aqui)
